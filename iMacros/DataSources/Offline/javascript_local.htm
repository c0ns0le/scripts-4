iMacros

Local Test Page

This page is the local version of the ONDIALOG iMacros Test Page. It demonstrates that iMacros can automate all browser based activities, regardless if they are online and offline. For example, you can automate tasks inside your companies Intranet or fill-out a HTML based router configuration dialog on your local PC. 

If you work on a PC without Internet connection, it is recommended that you check the box "Work Offline" in the Options dialog. This avoids the Internet connection check at Internet Macros startup.

If you are using Windows XP with Service Pack 2 and if you use this page inside of Internet Explorer you need to allow active content for this page in order to run the demo macro correctly. Or you simply run the macro inside of the stand-alone iMacros Browser (recommended).
